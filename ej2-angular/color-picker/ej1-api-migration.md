# Ej1 api migration in Angular Color picker component

This article describes the API migration process of ColorPicker component from Essential JS 1 to Essential JS 2.

## Properties

| Behavior | API in Essential JS 1 | API in Essential JS 2 |
| --- | --- | --- |
| Default |**property**: *value* <br/><br/> `<input ej-colorpicker id="colorpicker" value="#278787" />` | **property:** *value* <br/><br/> `<input ejs-colorpicker id="colorpicker" value="#278787" />` |
| Inline mode color picker | **property**: *displayInline* <br/><br/> `<input ej-colorpicker id="colorpicker" [displayInline]="true" />` | **property:** *inline* <br/><br/> `<input ejs-colorpicker id="colorpicker" [inline]="true" />` |
| Adding custom class | **property**: *cssClass* <br/><br/> `<input ej-colorpicker id="colorpicker" cssClass ="custom-class" />` | **property:** *cssClass* <br/><br/> `<input ejs-colorpicker id="colorpicker" cssClass="custom-class" />` |
| Disable the ColorPicker component | **property**: *enabled* <br/><br/> `<input ej-colorpicker id="colorpicker" [enabled]="false" />` | **property:** *disabled* <br/><br/> `<input ejs-colorpicker id="colorpicker" [disabled]="true" />` |
| To display custom text in button elements | **property**: *buttonText* <br/><br/> `<input ej-colorpicker id="colorpicker" [buttonText]="buttonText" />` <br/> buttonText: Object = { apply: "Apply", cancel: "Cancel", swatches: "Swatches" }; | Not Applicable |
 |To display customized text or content when mouse over the color picker elements | **property:** *tooltipText* <br/><br/> `<input ej-colorpicker id="colorpicker" [tooltipText]="tooltipText" />` <br/> tooltipText: Object = { switcher: "Switch",  currentColor: "New Color", selectedColor: "Old Color" }; | Not Applicable |
| Disable / hide opacity | **property**: *enableOpacity* <br/><br/> `<input ej-colorpicker id="colorpicker" [enableOpacity]="false" />` | **property:** *enableOpacity* <br/><br/> `<input ejs-colorpicker id="colorpicker" [enableOpacity]="false" />` |
| ColorPicker Button mode | **property**: *buttonMode* <br/><br/> `<input ej-colorpicker id="colorpicker" buttonMode="Dropdown" />` | Not Applicable |
| To show / hide the control (apply /  cancel) buttons | **property**: *showApplyCancel* <br/><br/> `<input ej-colorpicker id="colorpicker" [showApplyCancel]="false" />` | **property:** *showButtons* <br/><br/> `<input ejs-colorpicker id="colorpicker" [showButtons]="false" />` |
| To show / hide the clear button | **property**: *showClearButton* <br/><br/> `<input ej-colorpicker id="colorpicker" [showClearButton]="false" />` | Not Applicable |
| Show / hide the mode (picker / palette) switcher | **property**: *showSwitcher* <br/><br/> `<input ej-colorpicker id="colorpicker" [showSwitcher]="false" />` | **property:** *modeSwitcher* <br/><br/> `<input ejs-colorpicker id="colorpicker" [modeSwitcher]="false" />` |
| To show / hide the preview area | **property**: *showPreview* <br/><br/> `<input ej-colorpicker id="colorpicker" [showPreview]="false" />` | Not Applicable |
| To show / hide the recent selected color list | **property**: *showRecentColors* <br/><br/> `<input ej-colorpicker id="colorpicker" [showRecentColors]="true" />` | Not Applicable |
| To show / hide the color picker slider tooltip | **property**: *showTooltip* <br/><br/> `<input ej-colorpicker id="colorpicker" [showTooltip]="false" />` | Not Applicable |
| Custom icon in dropdown control color area | **property**: *toolIcon* <br/><br/> `<input ej-colorpicker id="colorpicker" toolIcon="e-font-icon" />` | Not Applicable |
| ColorPicker mode |**property**: *modelType* <br/><br/> `<input ej-colorpicker id="colorpicker" modelType ="Palette" />` | **property:** *mode* <br/><br/> `<input ejs-colorpicker id="colorpicker" mode="Palette" />` |
| Opacity value | **property**: *opacityValue* <br/><br/> `<input ej-colorpicker id="colorpicker" opacityValue=80 />` | Not Applicable |
| Number of columns in color palette | **property**: *columns* <br/><br/> `<input ej-colorpicker id="colorpicker" columns=10 />` | **property:** *columns* <br/><br/> `<input ejs-colorpicker id="colorpicker" columns=15 />` |
| Custom colors | **property**: *palette* <br/><br/>  `<input ej-colorpicker id="colorpicker" palette="CustomPalette" modelType ="Palette" [custom]="colors" />` <br/> colors: Array<*any*> = [ "ffffff", "ffccff",  "ff99ff", "ff66ff", "ff33ff", "ff00ff", "ccffff", "ccccff", "cc99ff", "cc66ff", "cc33ff", "cc00ff", "99ffff", "99ccff", "9999ff", "9966ff", "9933ff", "9900ff", "ffffcc", "ffcccc" ]; | **property:** *presetColors* <br/><br/> `<input ejs-colorpicker id="colorpicker" mode="Palette" [presetColors]="colors" />` <br/> colors: { [key: string]: string[] } = { <br/> &nbsp; 'custom': [ "ffffff", "ffccff",  "ff99ff", "ff66ff", "ff33ff", "ff00ff", "ccffff", "ccccff", "cc99ff", "cc66ff", "cc33ff", "cc00ff", "99ffff", "99ccff", "9999ff", "9966ff", "9933ff", "9900ff", "ffffcc", "ffcccc" ] <br/> }; |

| Localization | **property**: *locale* <br/><br/> ej.ColorPicker.Locale["zh-CN"] = { <br/> &nbsp; buttonText: { <br/> &nbsp;  &nbsp; apply: "应用", <br/> &nbsp;  &nbsp; cancel: "取消", <br/> &nbsp;  &nbsp; swatches: "色板" <br/> &nbsp; }, <br/> &nbsp; tooltipText: { <br/> &nbsp;  &nbsp; switcher: "切换器, <br/> &nbsp;  &nbsp; addButton: "添加颜色", <br/> &nbsp;  &nbsp; basic: "基本"<br/> &nbsp; } <br/> } <br/> `<input ej-colorpicker id="colorpicker" locale="zh-CN" />` | **property:** *locale* <br/><br/> L10n.load({ <br/> &nbsp; 'ar': { <br/> &nbsp;  &nbsp; "colorpicker": { <br/> &nbsp;  &nbsp;  &nbsp; "Apply":"تطبيق", <br/> &nbsp;  &nbsp;  &nbsp; "Cancel":"إلغاء", <br/> &nbsp;  &nbsp;  &nbsp; "ModeSwitcher": "مفتاح كهربائي الوضع" <br/> &nbsp;  &nbsp; } <br/> &nbsp; } <br/> }); <br/> `<input ejs-colorpicker id="element" locale="ar" />` |
| Right to left | **property:** *enableRTL* <br/><br/> `<input ej-colorpicker id="colorpicker" [enableRTL] ="true" />` |**property:** *enableRtl* <br/><br/> `<input ejs-colorpicker id="colorpicker" [enableRtl]="true" />` |
