# Ej1 api migration in Angular Maskedtextbox component

This article describes the API migration process of MaskEdit component from Essential JS 1 to Essential JS 2.

## Common

<!-- markdownlint-disable MD033 -->

| Behavior | API in Essential JS 1 | API in Essential JS 2 |
| --- | --- | --- |
| Adding custom class | **Property:** *cssClass* <br /><br />`<input ej-maskedit  inputMode="ej.InputMode.Text" maskFormat="9999" cssClass = "custom" />` | **Property:** *cssClass*<br /><br />`<ejs-maskedtextbox #mask cssClass = "custom" mask='9999'></ejs-maskedtextbox>` |
| Destroy editor | Not Applicable | **Method:** *destroy*<br /><br />`<ejs-maskedtextbox #mask mask="00-000"></ejs-maskedtextbox>`<br />`@ViewChild("mask") mask: MaskedTextBoxComponent;`<br />`mask.destroy();` |
| Disable the maskedit control | **Method:** *disable* <br /><br />`<input ej-maskedit  inputMode="ej.InputMode.Text" maskFormat="0000" value="1234" />`<br />`var maskObj = $(“#mask”).data(“ejMaskEdit”);`<br />`maskObj.disable();` | **Can be achieved using**<br/><br />`<ejs-maskedtextbox #mask mask="00-00" value="1234"></ejs-maskedtextbox>`<br />`@ViewChild("mask") mask: MaskedTextBoxComponent;`<br />`mask.enabled = false;` |
| Enable the maskedit control | **Method:** *enable*<br /><br />`<input ej-maskedit  inputMode="ej.InputMode.Text" maskFormat="0000" value="1234" />`<br />`var maskObj = $(“#mask”).data(“ejMaskEdit”);`<br />`maskObj.enable();` |  **Can be achieved using**<br/><br />`<ejs-maskedtextbox #mask mask="00-00" value="1234"></ejs-maskedtextbox>`<br />`@ViewChild("mask") mask: MaskedTextBoxComponent;`<br />`mask.enabled=true;` |
| Control state | **Property:** *enabled*<br /><br />`<input ej-maskedit  inputMode="ej.InputMode.Text" maskFormat="00-000" enabled="false" />` | **Property:** *enabled*<br /><br />`<ejs-maskedtextbox mask='00-000' enabled=false></ejs-maskedtextbox>` |
| Persistence | **Property:** *enablePersistence*<br /><br />`<input ej-maskedit  inputMode="ej.InputMode.Text" maskFormat="0000" enablePersistence="true" />` | **Property:** *enablePersistence*<br /><br />`<ejs-maskedtextbox mask='0000' enablePersistence=true></ejs-maskedtextbox>` |
| Triggers when editor is focused in | **Event:** *focusIn*<br /><br />`<input ej-maskedit  inputMode="ej.InputMode.Text" maskFormat="00-00" (focusIn)="focusIn()"/>`<br /><br />**script**<br />`focusIn() {}` | **Event:** *focus*<br /><br />`<ejs-maskedtextbox mask='0000' (focus) = "onFocus()"></ejs-maskedtextbox>`<br /><br />**script**<br />`public onFocus(): void {}` |
| Triggers when editor is focused out | **Event:** *focusOut*<br /><br />`<input ej-maskedit  inputMode="ej.InputMode.Text" maskFormat="0000" (focusOut)="focusOut()"/>`<br /><br />**script**<br />`focusOut() {}` | **Event** *blur*<br/><br/>`<ejs-maskedtextbox mask='0000' (blur) = "onBlur()"></ejs-maskedtextbox>`<br/><br />**Script**<br/>`public onBlur(): void {}`|
| Sets height | **Property:** *height*<br /><br />`<input ej-maskedit  inputMode="ej.InputMode.Text" maskFormat="0000" height="30px" />` | **Can be achieved using,** <br /><br />`<ejs-maskedtextbox #mask cssClass = "custom" mask='0000'></ejs-maskedtextbox>`<br />**Css**<br />`.e-maskedtextbox.custom{`<br />`height: 30px`<br />`}` |
| HTML Attributes | **Property:** *htmlAttributes*<br /><br />`<input ej-maskedit  inputMode="ej.InputMode.Text" maskFormat="0000" [htmlAttributes]= "htmlAttr" />`<br /><br />**script**<br />`constructor() {`<br />`this.htmlAttr = {name: "maskedtextbox"};`<br />`}` | **Can be achieved using,**<br/>`<ejs-maskedtextbox mask='999-99' value='12345' name="masked" ></ejs-maskedtextbox>` |
| Specifies Input mode | **Property:** *inputMode*<br /><br />`<input ej-maskedit  inputMode="ej.InputMode.Text" maskFormat="0000" />` | **Can be achieved using,**<br/>`<ejs-maskedtextbox mask='999-99' value='12345' type="password" ></ejs-maskedtextbox>` |
| Triggers on key press | **Event:** *keyPress*<br /><br />`<input ej-maskedit  inputMode="ej.InputMode.Text" maskFormat="0000" (keyPress)="onKeyPress()"/`><br /><br />**script**<br />`onKeyPress() {}` | **Can be achieved using**<br/>`<ejs-maskedtextbox mask='999-99'id="mask" value='12345' placeholder= "Masked TextBox" floatLabelType= 'Always' (created)="onCreate($event)"></ejs-maskedtextbox>`<br/>**Script**<br/>`public onCreate() {`<br/>`document.getElementById("mask").addEventListener('keypress',(event:any)=>{`<br/>`});`<br/>`}` |
| Triggers on key up | **Event:** *keyUp*<br /><br />`<input ej-maskedit  inputMode="ej.InputMode.Text" maskFormat="0000" (keyUp)="onKeyUp()"/`><br /><br />**script**<br />`onKeyUp() {}` | **Can be achieved using**<br/>`<ejs-maskedtextbox mask='999-99'id="mask" value='12345' placeholder= "Masked TextBox" floatLabelType= 'Always' (created)="onCreate($event)"></ejs-maskedtextbox>`<br/>**Script**<br/>`public onCreate() {`<br/>`document.getElementById("mask").addEventListener('keyup',(event:any)=>{`<br/>`});`<br/>`}` |
| Triggers on mouse out in maskedit control | **Event:** *mouseOut*<br /><br />`<input ej-maskedit  inputMode="ej.InputMode.Text" maskFormat="0000-000" (mouseOut)="onMouseOut()"/>`<br /><br />**script**<br />`onMouseOut() {}` | **Can be achieved using**<br/>`<ejs-maskedtextbox mask='999-99'id="mask" value='12345' placeholder= "Masked TextBox" floatLabelType= 'Always' (created)="onCreate($event)"></ejs-maskedtextbox>`<br/>**Script**<br/>`public onCreate() {`<br/>`document.getElementById("mask").addEventListener('mouseout',(event:any)=>{`<br/>`});`<br/>`}` |
| Triggers when mouse over in maskedit control | **Event:** *mouseOver*<br /><br />`<input ej-maskedit  inputMode="ej.InputMode.Text" maskFormat="00-00" (mouseOver)="onMouseOver()"/>`<br /><br />**script**<br />`onMouseOver() {}` | **Can be achieved using**<br/>`<ejs-maskedtextbox mask='999-99'id="mask" value='12345' placeholder= "Masked TextBox" floatLabelType= 'Always' (created)="onCreate($event)"></ejs-maskedtextbox>`<br/>**Script**<br/>`public onCreate() {`<br/>`document.getElementById("mask").addEventListener('mouseover',(event:any)=>{`<br/>`});`<br/>`}` |
| Name of maskedit control | **Property:** *name*<br /><br />`<input ej-maskedit  inputMode="ej.InputMode.Text" maskFormat="0000" name="pin"/>` | **Can be achieved using,**<br/>`<ejs-maskedtextbox mask='999-99' value='12345' name="masked" ></ejs-maskedtextbox>` |
| Triggers on keydown | **Event:** *onKeyDown*<br /><br />`<input ej-maskedit  inputMode="ej.InputMode.Text" maskFormat="9999-9999" (onKeyDown)="onKeyDown()"/>`<br /><br />**script**<br />`onKeyDown() {}` | **Can be achieved using**<br/>`<ejs-maskedtextbox mask='999-99'id="mask" value='12345' placeholder= "Masked TextBox" floatLabelType= 'Always' (created)="onCreate($event)"></ejs-maskedtextbox>`<br/>**Script**<br/>`public onCreate() {`<br/>`document.getElementById("mask").addEventListener('keydown',(event:any)=>{`<br/>`});`<br/>`}` |
| Read only | **Property:** *readOnly*<br /><br />`<input ej-maskedit  inputMode="ej.InputMode.Text" maskFormat="99-999" readOnly="true"/>` | **Can be achieved using**<br/>`<ejs-maskedtextbox #mask width="100px" mask='0000' enabled = false></ejs-maskedtextbox>`<br/>**CSS**<br/>`.e-mask{`<br/> `background-image: none !important;`<br/>`border-bottom-color:  rgba(0, 0, 0, 0.42) !important;` <br/>`}` |
| Right to left | **Property:** *textAlign*<br /><br />`<input ej-maskedit  inputMode="ej.InputMode.Text" maskFormat="9999" textAlign="Right"/>` | **Property:** *enableRtl*<br /><br />`<ejs-maskedtextbox #mask enableRtl=true mask='9999'></ejs-maskedtextbox>` |
| Sets width | **Property:** *width*<br /><br />`<input ej-maskedit  inputMode="ej.InputMode.Text" maskFormat="0000" width="100px"/>` | **Property:** *width*<br /><br />`<ejs-maskedtextbox #mask width="100px" mask='0000'></ejs-maskedtextbox>` |

| --- | --- | --- |

| Validation Rules | **Property:** *validationRules*<br /><br />`<input ej-maskedit  inputMode="ej.InputMode.Text" maskFormat="00-00"/>`<br /><br />**script**<br />`constructor() {`<br />`this.validationRules = {required: true};`<br />`}` | **Can be achieved using Form Validation**<br/>`<form #formElement class="form-horizontal">`<br />`<ejs-maskedtextbox id="masktextbox" #mask="" mask='000-000-0000' name="mask_value" placeholder= 'Mobile Number' floatLabelType= 'Always'>`<br/>`</ejs-maskedtextbox>`<br/>`</form>`<br/>**Script**<br/> `@ViewChild('formElement') element;` <br /> `@ViewChild('mask') mask: MaskedTextBoxComponent;`</br>`public formObject: FormValidator;`<br/> `let options: FormValidatorModel = {`<br/>`rules: {`<br/>`'mask_value': { required: [true] },`<br/>`}`<br/>`}`<br/>`this.formObject = new FormValidator(this.element.nativeElement, options);`<br/>`let customPlace: (element: HTMLElement, error: HTMLElement) => void = (element: HTMLElement, error: HTMLElement) => {`<br/>`document.querySelector(".form-group").appendChild(error);`<br/>`};`<br/>`this.formObject.customPlacement = customPlace;` |
