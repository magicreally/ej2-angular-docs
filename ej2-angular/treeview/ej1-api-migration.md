# Ej1 api migration in Angular Treeview component

This article describes the API migration process of TreeView component from Essential JS 1 to Essential JS 2.

## Add nodes

| Behavior | API in Essential JS 1 | API in Essential JS 2 |
| --- | --- | --- |
| Add node | **Method:** *addNode*<br/><br/>`<ej-treeview id='tree' [fields]='field'></ej-treeview>`<br /><br/>**Script**<br/>`var treeObj = $(“#tree”).data(“ejTreeView”);`<br />`treeObj.addNode("Node", "#book");` | **Method:** *addNodes*<br/><br/>`<ejs-treeview id='tree' [fields]='field'></ejs-treeview>`<br /><br />**Script**<br />`@ViewChild("tree") tree: TreeViewComponent;`<br />`var object = [{ id: "temp", name: "New node" }, { id: "new", name: "New node 1" }];`<br/>`tree.addNodes(object, "book");` |
| Triggers before adding node | **Event:** *beforeAdd*<br/><br/>`<ej-treeview id='tree' (beforeAdd)="beforeAdd()" [fields]='field'></ej-treeview>`<br /><br />**Script**<br />`beforeAdd() {}` | Not Applicable |
| Adding node after a particular node | **Method:** *insertAfter*<br/><br/>`<ej-treeview id='tree' [fields]='field'></ej-treeview>`<br /><br/>**Script**<br/>`var treeObj = $(“#tree”).data(“ejTreeView”);`<br />`treeObj.insertAfter("node", "book");` | **Can be achieved using**<br/><br/>`<ejs-treeview id='tree' [fields]='field'></ejs-treeview>`<br /><br />**Script**<br />`@ViewChild("tree") tree: TreeViewComponent;`<br />`var object = [{ id: "1", name: "node" }],`<br/>`var child = tree.element.querySelector('[data-uid="book"]');`<br/>`var parent = child.parentElement.closest('.e-list-item');`<br/>`var level = parseInt(parent.getAttribute('aria-level'))+1;`<br/>`var childNodes = Array.from(parent.querySelectorAll('.e-list-item.e-level-'+level))`<br/>`var index = childNodes.indexOf(child)`<br/>`tree.addNodes(object, "book", index+1);` |
| Adding node before a particular node | **Method:** *insertBefore*<br/><br/>`<ej-treeview id='tree' [fields]='field'></ej-treeview>`<br /><br/>**Script**<br/>`var treeObj = $(“#tree”).data(“ejTreeView”);`<br />`treeObj.insertBefore("node", "book");` | **Can be achieved using**<br/><br/>`<ejs-treeview id='tree' [fields]='field'></ejs-treeview>`<br /><br />**Script**<br />`@ViewChild("tree") tree: TreeViewComponent;`<br />`var object = [{ id: "1", name: "node" }],`<br/>`var child = tree.element.querySelector('[data-uid="book"]');`<br/>`var parent = child.parentElement.closest('.e-list-item');`<br/>`var level = parseInt(parent.getAttribute('aria-level'))+1;`<br/>`var childNodes = Array.from(parent.querySelectorAll('.e-list-item.e-level-'+level))`<br/>`var index = childNodes.indexOf(child)`<br/>`tree.addNodes(object, "book", index-1);` |
| Triggers when node is added successfully | **Event:** *nodeAdd*<br/><br/>`<ej-treeview id='tree' (nodeAdd)="nodeAdd()" [fields]='field'></ej-treeview>`<br /><br />**Script**<br />`nodeAdd() {}` | **Event:** *dataSourceChanged*<br/><br/>`<ejs-treeview id='tree' (dataSourceChanged)="dataSourceChanged()" [fields]='field'></ejs-treeview>`<br /><br />**Script**<br />`public dataSourceChanged(): void {}` |
